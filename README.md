# 🔍 SpyPhotoClean

**Εντοπισμός και εκκαθάριση παρόμοιων φωτογραφιών**

Το SpyPhotoClean είναι μια desktop εφαρμογή για Windows που εντοπίζει οπτικά παρόμοιες
ή διπλότυπες φωτογραφίες σε έναν φάκελο (και τους υποφακέλους του), τις ομαδοποιεί
και σας επιτρέπει να τις διαγράψετε με ασφάλεια — μέσω παραθύρου επιβεβαίωσης.

Χρησιμοποιεί τον αλγόριθμο **perceptual hashing (pHash)**, ο οποίος συγκρίνει
φωτογραφίες βάσει οπτικής ομοιότητας και όχι βάσει ονόματος αρχείου ή μεγέθους.

---

## 📋 Δυνατότητες

- **Αναδρομική σάρωση** φακέλου και υποφακέλων
- **Υποστηριζόμενοι τύποι**: JPG, JPEG, PNG, GIF, BMP, TIFF, WebP
- **Ρυθμιζόμενο κατώφλι ομοιότητας** (0–30)
- **Προεπισκόπηση thumbnails** με πληροφορίες αρχείου (μέγεθος, διαστάσεις, φάκελος)
- **Άνοιγμα φωτογραφίας** στην προεπιλεγμένη εφαρμογή προβολής
- **Οριστική διαγραφή** με παράθυρο επιβεβαίωσης
- **Real-time ενημέρωση** μπάρας προόδου κατά τη σάρωση
- **Ακύρωση** σάρωσης ανά πάσα στιγμή

## 🚀 Εκτέλεση

### Ως εκτελέσιμο (.exe)
Αν έχετε το αρχείο `SpyPhotoClean.exe`, απλά κάντε διπλό κλικ. Δεν χρειάζεται
εγκατάσταση Python ή οτιδήποτε άλλο.

### Από τον πηγαίο κώδικα
```bash
# Εγκατάσταση εξαρτήσεων
uv pip install flet Pillow imagehash

# Εκτέλεση
uv run main.py
```

## 🎯 Οδηγίες Χρήσης

### 1. Επιλογή Φακέλου
- Πληκτρολογήστε ή επικολλήστε τη διαδρομή του φακέλου στο πεδίο κειμένου.
- Πατήστε το εικονίδιο 📁 για να επιλέξετε φάκελο μέσω διαλόγου Windows.
- Πατήστε το εικονίδιο 🔗 για να ανοίξετε τον φάκελο στον Windows Explorer.

### 2. Ρύθμιση Κατωφλίου Ομοιότητας
Χρησιμοποιήστε το slider για να ρυθμίσετε πόσο «αυστηρή» θα είναι η σύγκριση:

| Τιμή   | Περιγραφή                          |
|--------|-------------------------------------|
| 0      | Μόνο πανομοιότυπα αρχεία           |
| 1–9    | Σχεδόν πανομοιότυπα                |
| 10–19  | Πολύ παρόμοια **(Προτεινόμενο)**   |
| 20+    | Χαλαρό φιλτράρισμα (πιθανά λάθη)  |

### 3. Σάρωση
Πατήστε **«Έναρξη Αναζήτησης»**. Η εφαρμογή θα:
1. Εντοπίσει όλες τις εικόνες στον φάκελο (αναδρομικά)
2. Υπολογίσει το perceptual hash κάθε εικόνας
3. Συγκρίνει τα hashes μεταξύ τους
4. Εμφανίσει τις ομάδες παρόμοιων φωτογραφιών

### 4. Διαχείριση Αποτελεσμάτων
- 👁️ **Προβολή**: Ανοίγει τη φωτογραφία σε πλήρη ανάλυση
- 🗑️ **Διαγραφή**: Διαγράφει **οριστικά** το αρχείο (με επιβεβαίωση)

> ⚠️ **Προσοχή**: Η διαγραφή είναι μόνιμη — τα αρχεία δεν πηγαίνουν στον Κάδο Ανακύκλωσης.

## 🔧 Αντιμετώπιση Προβλημάτων

### Σφάλμα `FileExistsError` κατά την εκκίνηση
Οφείλεται σε κατεστραμμένη cache του Flet. Λύση:
```bash
uv run python -c "import shutil; shutil.rmtree(r'C:\Users\Student\.flet', ignore_errors=True)"
```

### Η εφαρμογή δεν εμφανίζει αποτελέσματα
Βεβαιωθείτε ότι ο φάκελος περιέχει εικόνες σε υποστηριζόμενες μορφές
(JPG, PNG, GIF, BMP, TIFF, WebP).
